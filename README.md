# Tools:Q Animation Offset Shift

[English](README.en.md)

## 概要

View 3D / Pose Mode

ボーンを移動させた際に、アニメーショントラック上の全てのキーの位置/回転/拡大縮小をシフトして反映させるアドオンです。

## UI

- Animation Offset Shift Enable
  - チェックすると機能を有効にする。 **有効時はタイムラインのAuto Keying(オートキー)がオフになる。**
